# ESports At Oregon State University Website

## Instructions

Built using [deno](https://docs.deno.com/runtime/getting_started/installation) and [svelte](https://svelte.dev/).
- Use `deno i` to install dependencies.
- Use `deno task` to see scripts.

## Dev Server
[Deno](https://docs.deno.com/runtime/getting_started/installation) must be installed.
```
deno i
deno task dev
```
